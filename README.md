# Data-Analysis-Ecommerce-Performence-Dashboard
## Project Objective 
I developed a comprehensive project in Excel using public datasets from Kaggle about customer journey , creating several dashboards and tables to analyze the data. This process involved several stages, including data pre-processing, data cleaning, and data visualization.

## Dataset
- <a href="https://github.com/viviannabila/Data-Analysis-Ecommerce-Journey-Dashboard/blob/main/Customer%20journey%20Final%20with%20Dashboard.xlsx">E-commerce journey Dataset</a>

## Question's (KPIs)
- What is the total number of sessions?
- What is the total number of users?
- What is the total number of sessions with successful purchases?
- What percentage of visitors made a purchase?
- What is the average time spent on the page? 
- What is the average number of items in the cart? 
- What is the drop-off rate at each stage of the customer journey?
- What are the sales figures per country based on device type?
- Which sources are most effective at bringing in users?
- What is the distribution of visits based on device?
- How have purchases developed from month to month?
